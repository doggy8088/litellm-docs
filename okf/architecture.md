# Application shell

The website is a Docusaurus 3 application. `package.json` declares Docusaurus `3.8.1` and the React runtime, while `docusaurus.config.js` sets the production URL to `https://docs.litellm.ai/`, the default locale to English, warning behavior for broken links, custom CSS, and the theme configuration.

# Navigation and plugins

`sidebars.js` defines the main documentation navigation, including integrations, guardrails, alerting and monitoring, cost optimization, prompt management, AI tools, providers, proxy, tutorials, and other areas. `sidebars-release-notes.js` exposes release notes through an autogenerated sidebar.

The configuration also wires image optimization, ideal images, Mermaid, client redirects, optional Inkeep search and chat, raw-Markdown handling, Google Analytics in production, Crisp chat, and FeedbackRocket. The Inkeep integration is conditional on `INKEEP_API_KEY`, so the documentation site can build without that key.

# Publishing

The repository README states that deployment is handled automatically by Vercel when changes are pushed to `main`. The build output is the `build/` directory.
