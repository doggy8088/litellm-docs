# Purpose

`litellm-docs` is the source repository for [docs.litellm.ai](https://docs.litellm.ai), the documentation site for [LiteLLM](https://github.com/BerriAI/litellm). The site is built with Docusaurus 3 and documents the LiteLLM Python SDK, the self-hosted LLM Gateway / Proxy, provider integrations, observability, routing, caching, MCP, tutorials, and release notes.
