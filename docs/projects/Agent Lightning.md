# Agent Lightning {#agent-lightning}

[Agent Lightning](https://github.com/microsoft/agent-lightning) 是 Microsoft 的開源框架，用於透過強化學習、自動提示最佳化與監督式微調來訓練及最佳化 AI 代理程式，幾乎不需要修改程式碼。

它可搭配任何代理程式框架使用，包括 LangChain、OpenAI Agents SDK、AutoGen 和 CrewAI。Agent Lightning 在底層使用 LiteLLM Proxy 來路由 LLM 請求並收集追蹤資料，以支援其訓練演算法。
