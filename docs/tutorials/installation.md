# 設定環境 {#set-up-environment}

讓我們取得設定示範環境所需的必要金鑰。

每個 LLM 提供者都需要 API 金鑰（例如 `OPENAI_API_KEY`）。您可以從 OpenAI、Cohere 和 AI21 取得 API 金鑰，**無需等待名單**。

讓我們為我們的示範取得它們！

**OpenAI**: https://platform.openai.com/account/api-keys  
**Cohere**: https://dashboard.cohere.com/welcome/login?redirect_uri=%2Fapi-keys（不需要信用卡）  
**AI21**: https://studio.ai21.com/account/api-key（不需要信用卡）
